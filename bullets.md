# Tools Team Bullets

Generates a weekly bullet-point list of all the work done by ASF's Tools Team

![CI/CD All The Things!](docs/images/ci-cd-all-the-things.png)


## PyCharm integration

Queries can be run using the [JS GraphQL IntelliJ Plugin](https://jimkyndemeyer.github.io/js-graphql-intellij-plugin/).

The plugin [examples repository](https://github.com/jimkyndemeyer/graphql-config-examples) Github GraphQL v4 API config
provided the `.graphqlconfig` and `github-schema.graphql`.
